# PassiveR3con

`PassiveR3con` — русскоязычная программа для **пассивной OSINT-разведки** по доменам и поддоменам с современным web UI.

## Что умеет

- Принимает одну цель (`--target`) или файл (`--file`) в CLI-режиме.
- Имеет **современный красивый web UI** (`--ui`) для запуска сканирований из браузера.
- **Реально запускает инструменты**, а не просто выводит команды:
  - `whois`
  - `dig`
  - `nslookup`
  - `host`
- Дополнительно использует пассивные источники:
  - DNS-резолвинг через `socket.getaddrinfo`
  - CT-логи через `crt.sh`
- Сохраняет результаты в SQLite БД (`scans`, `findings`).
- Генерирует современный HTML-отчёт на русском языке для каждого сканирования.

## Быстрый старт

### CLI режим

```bash
python3 passive_osint_ru.py --target example.com
```

или

```bash
python3 passive_osint_ru.py --file targets.txt
```

### Web UI (рекомендуется)

```bash
python3 passive_osint_ru.py --ui --host 127.0.0.1 --port 8080
```

После запуска откройте: `http://127.0.0.1:8080`

## Аргументы

- `--target` — один домен/поддомен
- `--file` — файл с доменами/поддоменами
- `--db` — путь к SQLite БД (по умолчанию `passive_osint.db`)
- `--report-dir` — каталог для отчётов (по умолчанию `reports`)
- `--ui` — запуск web UI
- `--host` — хост UI (по умолчанию `127.0.0.1`)
- `--port` — порт UI (по умолчанию `8080`)

## Пример `targets.txt`

```text
example.com
sub.example.com
# комментарии поддерживаются
```
